# TriboPersonalEstudio
Aplicação para academia onde o proprietário pode cadastrar alunos, exercícios, série de exercícios, como acompanhar dados do aluno, como
data de inscrição, validade do contrato.

> Status: Em Desenvolvimento ⚠️
